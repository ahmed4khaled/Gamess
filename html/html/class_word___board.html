Detailed Description

Implements a 3×3 word-forming game where players place letters.

Game Rules:

  • Board is 3×3 and starts empty ('-').
  • A player places a letter (A–Z) each turn.
  • After a placement, all lines passing through that cell (row, column, diag, anti-diag) are checked.
  • If any line forms a valid 3-letter word from the dictionary, the current player immediately wins.
  • If the board becomes full with no word formed, the game is a draw.

Dictionary:

  • Loaded from dic.txt.
  • Only 3-letter uppercase words are accepted.

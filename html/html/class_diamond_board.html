Detailed Description

Implements the board for Diamond Tic-Tac-Toe (5x5 diamond shape).

Only cells that satisfy |r - 2| + |c - 2| <= 2 are playable. Win condition: player must form one 3-in-a-row AND one 4-in-a-row in different directions.

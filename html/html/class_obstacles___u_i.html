Detailed Description

UI handler for the 5×5 Obstacles Tic-Tac-Toe game.

Responsibilities:

  • Display the board
  • Ask for human moves
  • Provide simple AI: • Try to win • Try to block • Otherwise random empty move
  • Insert obstacles before game starts

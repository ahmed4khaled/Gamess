\doxysubsection{Detailed Description}
Standard 3×3 Tic-\/\+Tac-\/\+Toe board. 

Rules\+:
\begin{DoxyItemize}
\item \doxylink{class_board}{Board} size\+: 3×3
\item Symbols\+: X or O
\item A player wins by forming 3-\/in-\/a-\/row in any direction.
\item Draw\+: board is full and no win.

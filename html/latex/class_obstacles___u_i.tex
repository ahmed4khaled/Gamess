\doxysubsection{Detailed Description}
\doxylink{class_u_i}{UI} handler for the 5×5 Obstacles Tic-\/\+Tac-\/\+Toe game. 

Responsibilities\+:
\begin{DoxyItemize}
\item Display the board
\item Ask for human moves
\item Provide simple AI\+: • Try to win • Try to block • Otherwise random empty move
\item Insert obstacles before game starts 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Implements the board for Diamond Tic-\/\+Tac-\/\+Toe (5x5 diamond shape). 

Only cells that satisfy \texorpdfstring{$\vert$}{|}r -\/ 2\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}c -\/ 2\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<}= 2 are playable. Win condition\+: player must form {\bfseries{one 3-\/in-\/a-\/row AND one 4-\/in-\/a-\/row}} in different directions.

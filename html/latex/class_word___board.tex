\doxysubsection{Detailed Description}
Implements a 3×3 word-\/forming game where players place letters. 

Game Rules\+:
\begin{DoxyItemize}
\item \doxylink{class_board}{Board} is 3×3 and starts empty (\textquotesingle{}-\/\textquotesingle{}).
\item A player places a letter (A–Z) each turn.
\item After a placement, all lines passing through that cell (row, column, diag, anti-\/diag) are checked.
\item If any line forms a {\itshape valid 3-\/letter word} from the dictionary, the current player immediately wins.
\item If the board becomes full with no word formed, the game is a draw.
\end{DoxyItemize}

Dictionary\+:
\begin{DoxyItemize}
\item Loaded from {\ttfamily dic.\+txt}.
\item Only 3-\/letter uppercase words are accepted. 
\end{DoxyItemize}

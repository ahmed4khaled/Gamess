\doxysubsection{Detailed Description}
Implements Misère Tic-\/\+Tac-\/\+Toe (avoid making 3-\/in-\/a-\/row). 

Rules\+:
\begin{DoxyItemize}
\item Standard 3×3 grid.
\item Players place X or O normally.
\item The player who {\bfseries{forms}} a 3-\/in-\/a-\/row {\bfseries{loses}}, not wins.
\item If the board becomes full without anyone forming 3-\/in-\/a-\/row, the game is a draw. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Implements Numerical Tic-\/\+Tac-\/\+Toe game logic. 

Rules\+:
\begin{DoxyItemize}
\item \doxylink{class_board}{Board} is 3×3.
\item \doxylink{class_player}{Player} 1 uses ODD numbers\+: \{1,3,5,7,9\}
\item \doxylink{class_player}{Player} 2 uses EVEN numbers\+: \{2,4,6,8\}
\item A number cannot be reused.
\item Win\+: any row/column/diagonal summing to EXACTLY 15. 
\end{DoxyItemize}

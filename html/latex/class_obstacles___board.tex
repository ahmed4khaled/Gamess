\doxysubsection{Detailed Description}
Tic-\/\+Tac-\/\+Toe with obstacles placed on a 5×5 grid. 

Game Rules\+:
\begin{DoxyItemize}
\item \doxylink{class_board}{Board} size\+: 5×5
\item Empty cells = blank symbol
\item Obstacles = obstacle symbol (cannot be overwritten)
\item \doxylink{class_player}{Player} wins by forming {\bfseries{3-\/in-\/a-\/row}} horizontally, vertically, or diagonally.
\item Obstacles block lines and block placement.
\item Draw\+: board full with no winner.
\end{DoxyItemize}

Important\+:
\begin{DoxyItemize}
\item Obstacles are pre-\/inserted randomly or fixed depending on \doxylink{class_u_i}{UI} logic.
\item Players place X or O in valid (non-\/obstacle, empty) cells. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
User Interface class for the Word Game (3×3 letter placement). 

Responsibilities\+:
\begin{DoxyItemize}
\item Reading moves from human players.
\item Displaying the board in a readable format.
\item Providing simple AI behavior\+: • Try immediate win. • Try blocking opponent’s immediate win. • Choose heuristic best move. 
\end{DoxyItemize}

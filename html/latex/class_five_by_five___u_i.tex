\doxysubsection{Detailed Description}
Handles all user interaction for the 5×5 Tic-\/\+Tac-\/\+Toe game. 

This \doxylink{class_u_i}{UI} collects player moves, prints the board, and creates players. Also contains simple AI logic for computer players.
